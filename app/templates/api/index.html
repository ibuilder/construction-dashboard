Construction Dashboard API

Welcome to the Construction Dashboard API documentation. This API provides access to project data for mobile and external applications.

API Overview

The API provides RESTful endpoints for accessing project information, including:

  • Projects
  • Engineering documents (RFIs, Submittals)
  • Field reports (Daily Reports, Photos)
  • Safety observations and incidents
  • Cost management (Change Orders, Invoices)
  • Document verification via blockchain
For detailed API documentation with request/response examples, please visit our Swagger UI Documentation.

Authentication

API requests require authentication using a Bearer token. To obtain a token: 

POST /api/auth/token
Content-Type: application/json

{
    "email": "your.email@example.com",
    "password": "your_password"
}

The response will contain your token: 

{
    "status": "success",
    "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
    "expires": "2025-04-09T15:30:45.123456",
    "user": {
        "id": 1,
        "name": "John Doe",
        "email": "john.doe@example.com",
        "role": "General Contractor"
    }
}

Include this token in all API requests: 

GET /api/projects
Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...

Getting Started

To start using the API:

  1. Obtain an authentication token
  2. List available projects with GET /api/projects
  3. Access project details with GET /api/projects/{id}
  4. Explore other endpoints as needed
